# GymFlow

A minimal SaaS application built with Next.js, TypeScript, and Prisma.

## Getting Started

```bash
npm install
npm run db:push
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Structure

- `app/` — Next.js App Router pages and API routes
- `lib/` — Shared utilities and project config
- `prisma/` — Database schema

## API

- `GET /api/health` — Health check endpoint
